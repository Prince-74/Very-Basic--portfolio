# Cinematic Portfolio Hero

A Next.js App Router portfolio landing page with a sticky cinematic video intro, duplicated ambient video layer, GSAP entrance choreography, and an optimized Three.js bokeh particle canvas.

## Video asset

Place the provided talking-head video at:

```txt
public/talking-head.mp4
```

The hero component uses this file as both the foreground video and blurred ambient background layer.

## Run locally

```bash
npm install
npm run dev
```
