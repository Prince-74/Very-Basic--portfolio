# Prince Goyal — Portfolio

A Next.js portfolio with a cinematic video hero, GSAP entrance animations, Three.js bokeh particles, and full profile sections (About, Tech Stack, Projects, Experience, GitHub analytics, and more).

## Video asset

Place your talking-head video at:

```txt
public/talking-head.mp4
```

## Customize content

Edit all profile copy, links, and GitHub username in:

```txt
data/portfolio.ts
```

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Sections

1. Cinematic hero (video, Open to Work badge, tech tags, CTAs)
2. About · Profile
3. Tech Stack
4. AI / ML Expertise
5. Featured Projects (collapsible)
6. Experience
7. Achievements
8. Certifications
9. Coding Profiles
10. GitHub Analytics, Trophies, Activity Graph, Contribution Snake
11. Current Focus
12. Connect
13. Footer
